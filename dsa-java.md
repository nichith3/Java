# DSA - Java

## Stacks
```java
// Stack<type> s1 = new Stack<>();
// Stack s2 = new Stack();

Stack<Integer> s1 = new Stack<>();

s1.push(98);
s1.empty();       // false
s1.peek();        // top - 98
s1.search(98);    // index - 0
s1.pop();         
s1.size();
```
